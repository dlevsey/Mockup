How to Process Tips in X-Product

Follow the steps below to process tips using the Transaction Adjust Details report in X-Product.

Step 1: Activate Tip Option

  1. Hover over the Admin menu.

  2. Select Payment Preferences from the drop-down menu.

  3. Find the Activate Tip Option under the Settings column.

  4. Check the Activate box associated with the Activate Tip Option.

  5. Click the Save button below the table.

Step 2: Fill Out Transaction Adjust Details Report

  1. Hover over the Main Menu.

  2. Select Transaction Adjust from the drop-down menu.

  3. Search for the correct transaction.

  4. Find the Tip Amount column.

  5. Enter the authorized tip amount into the chosen transaction.

  6. Click Submit.
